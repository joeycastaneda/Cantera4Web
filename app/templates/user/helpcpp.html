Using C++

The C++ functionality is somewhat limited, but all Cantera functionality is available. Select "C++" from the language dropdown above the editor to use C++.

Formatting C++ code

Since the using C++ is somewhat limited, your code has to follow a simple guideline: pretend you're writing everything inside main()!

The try and catch blocks are simply there to output any Cantera specific errors to the user.
